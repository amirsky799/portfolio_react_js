## Installation

_For run this website you must follow the section below._

1. NPM
    ```sh
    npm start
    ```
    
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
  
3. Run Build
    ```sh
    npm run build
    ```
